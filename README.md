# woole-challengetech
Desafio técnico proposto pela Woole.

### Linguagens utilizadas:
  - HTML5
  - Javascript (coffee-script)
  - CSS3 (Sass) 

### Dependências do projeto: 
  - Angular JS (1.2.8)  
  - Angular UI Router (0.2.15)
  - Angular NG Resource	(1.5.1)
  - Google Maps JavaScript API Version 3
  - GeoXML3 (Renders KML)
